# ESLint OCI

## How to Install

1. add `eslint` and `eslint-config-oci` into your `devDependencies`

    ```json
      "devDependencies": {
        ...,
        "eslint": "^8.27.0",
        "eslint-config-oci": "https://github.com/orangecapinnovative/eslint-config-oci.git#v0.3.0"
      }
    ```

2. install them

    ```bash
      npm install
    ```
